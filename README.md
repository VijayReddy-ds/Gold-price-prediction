# Gold Price Prediction

This project focuses on analyzing and predicting gold prices using Python.

## Overview
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualization of trends
- Basic prediction model

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Outcome
- Identified trends in gold prices
- Built a simple predictive model
